API Reference

Use these endpoints to interact with the service.

GET/health
curl http://localhost:3000/health
POST/send-sms
curl -X POST http://localhost:3000/send-sms -H "Content-Type: application/json" -d '{"destinationAddr":"+251914558019","shortMessage":"Hello"}'
GET/send-sms?destinationAddr=...&shortMessage=...
http://localhost:3000/send-sms?destinationAddr=+251914558019&shortMessage=Hello

Request body

{ "destinationAddr": "+251914558019", "shortMessage": "Hello", "sourceAddr": "6301", "host": "10.204.181.70", "port": 5019, "systemId": "6301", "password": "your-password" }
